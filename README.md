Torus Black Hole

# Controls
ESC: quit

w: go foward

a: go left

s: go behind

d: go right

z: increase radius

x: decrease radius

c: increase radius

v: decrease radius

b: increase height of half-plane model

n: decrease height of half-plane model

j: switch texture

p: reset

# Requirements
- Windows
- Nvidia GPU
- 64-bits architecture
